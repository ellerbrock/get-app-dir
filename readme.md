## Project Status

### install with npm

`npm install --save get-app-dir`

### install with yarn

`yarn add get-app-dir`

### Example

#### ES5

```javascript
var getAppDir = require('get-app-dir')

console.log(getAppDir)
```

#### ES6

```javascript
import getAppDir from 'get-app-dir'

console.log(getAppDir)
```
